1. **EC2（Elastic Compute Cloud）**
    - ➤ 這是 AWS 的虛擬機服務，類似一台雲端的電腦。
    - ➤ 自動化腳本的應用可能包括：
        - ◦ 啟動或關閉 EC2 實例
        - ◦ 部署程式或設定環境（可能透過 shell script、Ansible、Terraform、CloudFormation 等）
        - ◦ 自動擴展、備份或監控

2. **S3（Simple Storage Service）**
    - ➤ 這是用來存放檔案（如圖片、備份檔、日誌檔）的雲端儲存服務。
    - ➤ 自動化可能包含：
        - ◦ 上傳/下載檔案
        - ◦ 定期清除特定檔案
        - ◦ 設定 bucket 權限或生命周期管理

3. **DynamoDB**
    - ➤ AWS 提供的 NoSQL 資料庫，適合高效能讀寫與擴展。
    - ➤ 自動化可能包含：
        - ◦ 建立或修改資料表
        - ◦ 匯入或匯出資料
        - ◦ 設定備份、指標監控、自動擴展
