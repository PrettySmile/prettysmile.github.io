# 各代碼意思
### 2xx 成功回應

- 200
    - 成功

### 3xx 重定向訊息

- 304
    - 未修改
    - (用戶端要求該網頁時，其內容並沒有變更，用戶端取本地快取(Local Cache)即可。)

### 4xx 用戶端錯誤回應

- 400
    - 參數錯誤。
    - 參數不符合規定，缺少參數、型態錯誤、缺少 header 等等。
- 401
    - 登入失敗
- 403
    - 權限有問題。
- 404
    - 找無此資源 或 路徑錯誤。

### 5xx 伺服器錯誤回應

- 500
    - 伺服器錯誤
    - 發生Exception
