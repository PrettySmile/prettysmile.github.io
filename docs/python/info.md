# Python 後端框架~ FastAPI 學習

## 選擇 FastAPI 學習的理由：
1. 適合用於 Web API 的輕量級框架。
2. 性能快。

## 比較

| 名字 | Node.js | Python |
| ---- | ---- | ------ |
| 特性 | 快速IO,事件驅動 | 語法非常簡潔 |
| 適合領域 | Web API | AI/ML |
| 執行序 | 單執行序 | 多執行序 |
| 生態系 | 適合 I/O 密集型 | 適合 CPU 密集型 |
