# Visual Studio Code 指定 JAVA 版本


1. 安裝套件 ```Extension Pack for Java``` 。
2. 輸入指令 ```CTRL+SHIFT+P``` 搜尋 ```configure java runtime``` 。
    ![alt text](../../assets/images/vsCode/image.png)

3. 設定 ```JDK Runtime``` (找到本機位置 JDK 安裝)。
    ![alt text](../../assets/images/vsCode/image-1.png)
